What is JQuery?

JQuery is a JavaScript library. It basically takes things that are commonly done in JavaScript but require a lot of JS code and rewrites them. JQuery is still using JavaScript to achieve everything, but it puts that code into snippets that you can use without worrying about the behind the scenes work.

What do they mean by a library?

A library is a set of pre-written Javascript code snippets that are used to speed up the development process. Like said above, JQuery uses JavaScript to create easier to use code snippets that accomplish different things. Some of the things that libraries make easier are AJAX calls and DOM manipulation.

What are some of the advantages of JQuery?

  • Way less code to accomplish cool things
  • Easier DOM traversal and manipulation
  • Easier event handling
  • Pre-made functions to accomplish cool animations
  • Pre-made widgets, like date pickers
  • Widely used, good cross-browser support

What are some of the disadvantages of JQuery?

  • Learning curve
  • Lots of reading API documentation to get a good understanding
  • Tons of functions, meaning you have to know what to Google
  • Need a CDN or a local version of the library to use
  • If JQuery doesn't have a solution to your problem, you have to write it yourself

List and describe three or more examples of JQuery libraries?

1. JQuery UI
JQuery UI focuses on, you guessed it, the user interface side of things. It offers widgets, interactions like drag and drop, and special effects to help you create a better user experience. All of their widgets are fully customizable to fit the color scheme or themes of your website, but there are also pre-built styles if you are looking purely for the widget's functionality.
2. JQuery Mobile
JQuery Mobile is a library that helps you create websites and app that are mobile friendly. It takes the guess work out of making sure your website or app is responsive. JQuery Mobile provides the same services as JQuery, so widgets and the like, because it is built on JQuery. Basically it is JQuery optimized for responsive design.
3. QUnit
QUnit is a testing library built on JQuery. It provides easy to use JavaScript based tools to create tests, and handles all things testing, like assertions and callbacks, with similar to JQuery syntax. If you are already familiar with JQuery, beggining the use of QUnit would be an easy transition into unit testing.

How does JQuery relate to and interact with Javascript?

JQuery is literally JavaScript code, just compartmentalized into functions and objects. When you use JQuery, you are still using JavaScript, and you often have to write your own JavaScript code(just a little bit) to utilize the full power of its functions and objects.
