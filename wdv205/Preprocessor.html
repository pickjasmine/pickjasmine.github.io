CSS Preprocessors

What does a CSS Preprocessor do?

Preprocessors extend CSS to include new abilities, such as variables and mixins, which make styling large projects much easier. Preprocessors take scss files and transform them into css files that are readable by all web browsers.

What are the advantages of using a CSS Preprocessor?

Easier to manage large amounts of CSS files, little learning curve from original CSS (if you use SCSS syntax) , can easily add mixins that will make it easier to apply CSS3 styles that can be seen on all browsers.

What are the disadvantages or issues you need to consider when using a CSS Preprocessor?

Learning a new "language" (not really a whole language, but the syntax), learning curve for using the command prompt
