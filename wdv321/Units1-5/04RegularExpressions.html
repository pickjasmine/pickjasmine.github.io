Regular Expressions in JavaScript and jQuery

What commands in Javascript are used to process Regular Expressions? Please list the commands and describe their functionality.

regexp.exec(string)
Does a search through the string, and returns an array of values if any matches are found. Returns null is there are no matches.
regexp.test(string)
tests the string and sees if it matches the regex, returning a boolean
string.match(regexp pattern)
searches string for a match of the pattern, returning an array of the results. You must include the gloabl flag (g) if you want to have more than one result in the array.
string.search(regexp pattern)
searches string for a match of the pattern, returning beginning index of the match if it is found. Returns -1 if no matches are found.
string.replace(regexp pattern, string)
searches string for a match of the pattern, replacing the entire match with the string parameter
string.split(regexp pattern)
searches string for a match of the pattern, dividing the string into an array at every pattern match.

What do they mean when they use the term 'pattern' with Regular Expressions?

The patterns in regular expressions are used to detect strings. The patterns are made up of symbols and characters that outline a "pattern" that the string must match in order to match the regex.
