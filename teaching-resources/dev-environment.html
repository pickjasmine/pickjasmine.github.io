Setting Up

Your Development Environment


Before you can run our first app, you must have NodeJS and Yarn installed.

Windows users should use the git client

First Step: SSH

  • Check for existing SSH Keys

  • Generate SSH Key

  • Add SSH Key to Your Computer

  • Add SSH Key to Your Github Account

Second Step: Clone this repo

  • Make sure you know where you are in your file system before continuing.

  • type git clone git@github.com:pickjasmine/movies-fall2019.git into your terminal

Third Step: Run the app

  • From the same place you just cloned the repository...

  • type cd movies-fall2019 and press enter

  • type yarn && yarn start and press enter


If everything worked, a browser tab should open with the application running. If you see a black and white gradient background, you are good to go!
