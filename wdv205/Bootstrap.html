Bootstrap

What is Bootstrap?

Bootstrap is a framework used for making responsive websites. Bootstrap comes with pre-built items, such as nav bars, footers, cards, and grids that make creating a website that is responsive and interesting very easy to make. Bootstrap also has themes that make it easy for people with little to no development background to make websites that are aesthetically appealing and fully functional.

What is a CSS framework?

A CSS framework is a software that creates websites. Frameworks make creating websites that follow internet standards and regulations easy and fast for people of all skill level. Using frameworks allows the web designer to focus on the content and customization of the site, rather than dealing with how to get the visual functionality that they want.

Why is Bootstrap referred to as a framework?

Bootstrap is a framework, because it does what has been described above. Bootstrap makes standards-compliant websites and provide a base for anyone who is looking to create a responsive website without needing to code everything yourself. If you have a basic knowledge of HTML and CSS, or if you have the capacity to learn a little bit about them, you will be able to utilize Bootstrap to create beautiful and responsive web pages.

Advantages/Disadvantages of a framework

Advantages

  • No reinventing the wheel
  • Lots of frameworks to choose from
  • Pre-built themes/real life examples for inspiration
  • No need to be creatively inclined
  • Faster than making websites yourself

Disadvantages

  • Learning curve
  • Customization is limited for some frameworks
  • Using themes leads to repetitive designs
  • Framework might not have every feature you want
